About the studio

Built small.
Built to last.

Rigour Labs is a small team product studio building AI tools for consulting and advisory professionals. The aim is simple: software that makes expert thinking sharper, faster, and more defensible.

Every product comes from a real problem in the workflow. If it would not survive a board paper review, it does not ship.

The studio is based in Sydney, Australia.

01
Defensible over polished

Generic outputs are the enemy. Every tool is built to produce work you can stand behind in a meeting, not work that merely looks correct.

02
Context compounds

The best AI tools get better the more you use them. Memory, profiles, and project context are first-class features, not afterthoughts.

03
One product at a time

Nothing ships until it solves the problem properly. The backlog is long but the focus is narrow.

04
Privacy by default

No tracking scripts. No advertising. Data stays in Australia. Built for professionals who handle sensitive client information.
